# Sound Effects

These MP3 files are **silent placeholders** — each contains one frame of MPEG silence so the app bundles correctly. Replace them with real audio files (keep the same filenames and ideally keep files under 50 KB each).

| File | When it plays |
|---|---|
| `whoosh.mp3` | New question card appears on Play screen |
| `cheer.mp3` | Streak milestone (every 3 consecutive dares) |
| `drumroll.mp3` | Handoff screen (plays while waiting for player) |

Suggested sources: [freesound.org](https://freesound.org), [mixkit.co](https://mixkit.co).
